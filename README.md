## Deployment

Each folder contains a static build — deploy directly to Cloudflare Pages or any static host.

To deploy all demos at once, consider a multi-site Cloudflare Pages setup or a monorepo with subfolders.

## Archetypes

- **Editorial / Upscale** — Refined, restrained, journal-style
- **Southern / Heritage** — Warm, familiar, porch-welcoming
- **Bold / Fast-Casual** — Punchy, order-first, energetic
- **Coastal / Polished** — Airy, food-forward, destination-feeling
- **Family / Comfort** — Trust-first, practical, approachable

## Notes

- Branding kept conservative during demo phase (text-based, no copied logos)
- Easy to swap in real logo, photos, menus, and brand colors after hire
- All demos built with Astro static output
